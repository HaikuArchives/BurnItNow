BurnItNow burns data, audio and ISO images on CD, and blanks rewritable CD-RW media. It is a graphical frontend to the commandline tools suite cdrtools by JÃ¶rg Schilling et al.

BurnItNow's window consists of three parts: the menu bar, the session settings and several project modes in tabs at the bottom:

The Menus

The available menu items are few and easy to understand.
Under App you can open the About... panel and Quit the application.

Tools & settings provides options to clear images and folders that are created while using BurnItNow. They are located at /system/cache/, in case you want to have a look or save an image for later use.
Clear cache on quit is a setting that can be toggled with a checkmark. Activated, it automatically clears the cache every time you quit BurnItNow.
Clear cache now does this 'manually' when you select it.

Under Help you find the Usage instructions you're currently reading and can visit the Project website, for example to report an issue.

The Session Settings

The Session settings at the top of the window apply to all the different project modes described further down (besides blanking a CD-RW which doesn't need all those settings).

Checkboxes

The features of the top two checkmarks MultiSession and On-the-fly are currently not implemented and are therefore grayed out.
Simulation is a dummy mode that goes through all the motions without actually burning anything onto the disc.
Eject after burning does just that. A good way to remind you your CD's finished while you're busy elsewhere.

Speed slider

With the Burn speed slider you set the speed used when writing to the disc. The 'Min' and 'Max' settings are determined from the used media and your CD burner model by the "cdrecord" backend.
For best compatibility with stand-alone CD players, a speed of 4x is recommended.

When cloning, the disc is read out with a speed of 10x. In our tests, increased speed resulted in ever more read errors.

Popup menus

Below checkmarks and speed slider are two popup menus:
First is the recording mode. It's either Disc At Once (DAO) (actually "Session at once" (SAO)) where the burning is done in one go for the whole disc and it's fixated at the end. With Track At Once (TAO) the laser is turned off after every track and the disc isn't closed automatically.
With BurnItNow's current feature set, you're probably good with always using DAO.

Project Modes

Below the session settings are the different project modes of BurnItNow sorted in tabs. Each mode shows a number of buttons and a bold status line suggesting the next step for burning your project. Below that is a large area where cdrecord and the other commands working in the background print out their messages.

Data

In Data mode, you first use Choose folder to select the folder that's to be burnt to disc. That folder has to be set up beforehand by copying all the files and folders to it. The folder's name will be used as the label for the disc.
Next you click Build image to create an ISO image of that folder. Depending on the number of files and their size and the speed of your harddisk that may take some seconds.
Lastly, you check your session settings one more time, insert a blank disc, and click Burn disc to start the writing.

Audio

To burn an audio CD, you first drag & drop all tracks into the view to the right. These have to be WAV files! Now you can re-order them by dragging them around until you're satisfied. Remove tracks by selecting them and either press DEL or right-click and choose Remove.
You can change the ratio between the info view to the left and the tracks view to the right by dragging the splitter handle between them around. You can even collapse one of the views completely.

For best compatibility to stand-alone players, select a speed of 4x.
Finally, check your session settings once more, insert a blank disc and click Burn disc.

Image file

To write an ISO image file to a CD, you click Choose image and select the image to burn. Then check your session settings, insert a blank disc and click Burn disc.

Blank CD-RW

You start by choosing the type of blanking from the popup menu. Not all devices support all types. You may have to choose All which is slowest. From the cdrecord documentation:

AllBlank the entire disk. This may take a long time.
FastMinimally blank the disk. This results in erasing the PMA, the TOC and the pregap.
SessionBlank the last session.
Track Blank a track.
Track tailBlank the tail of a track.
UnreserveUnreserve a reserved track.
UncloseUnclose last session.

Now insert the CD-RW to blank and click Blank disc. Keep an eye on the output in the text area below to see what cdrecord had to say of your choice of blanking type.

Clone

This mode is for making a 1:1 copy of a CD.
You start by inserting the disc to clone and click on Create image. The disc will be ejected when its completely read out (at a fixed 10x speed). Now insert a blank disc, check your session settings and click Burn image.

Tips & Tricks

  • You can check if your project fits onto a CD with Haiku's DiskUsage. Just open the folder (or folder with the WAVs) to burn, right-click and choose Add-ons | DiskUsage from the context menu.

  • Data and Image mode: You can drag & drop the folder/image to burn instead of using the Choose folder/image button that opens the file dialog.

  • Data and Image mode: Once a folder/image is selected, clicking on its path to the left of the buttons opens its location in Tracker.

  • Audio mode: You can drag & drop folders with WAVs too.

  • Audio mode: Use a speed of 4x for best compatibility with stand-alone players.

Download

BurnItNow is directly available through HaikuDepot from the HaikuPorts repository. You can also build it yourself using Haikuporter. The source is hosted at GitHub.
